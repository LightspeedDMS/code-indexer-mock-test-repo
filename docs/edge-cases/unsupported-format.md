![Unsupported BMP](../../images/unsupported.bmp)

## Supported Formats Reference

Only these formats are supported:
- PNG (image/png)
- JPEG (image/jpeg)
- WebP (image/webp)
- GIF (image/gif)

BMP, TIFF, SVG, and other formats are not supported.
